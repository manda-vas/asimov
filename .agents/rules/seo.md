# SEO

- Every page must export `metadata` with `title` and `description`
- Use semantic HTML: `<header>`, `<main>`, `<section>`, `<footer>`, `<nav>`
- Only one `<h1>` per page — must contain the main keyword
- Heading hierarchy must be respected: h1 → h2 → h3, never skip levels
- Every `<Image>` must have a descriptive `alt` attribute, never empty or generic
- Above-the-fold images must use `priority` on `<Image>`
- Use `<Link>` for internal navigation — never `<a>`
