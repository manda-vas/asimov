# Hero Section — Curso de Python | Parte 2

## Como rodar
```bash
cd parte-2 && npm install && npm run dev
```
Abra o navegador na porta indicada no terminal.
Acesse [http://localhost:3000](http://localhost:3000)

### Ferramentas de IA utilizadas:
- **Stitch/Nano:** Geração de ativos visuais, UI Layout e prototipagem de ideias.
- **Antigravity:** Engenharia de prompts, estruturação, componentes e animações.

### Etapas de maior auxílio da IA
A IA foi o principal acelerador na **exploração estética e prototipagem rápida**. Em segundos, permitiu testar diferentes direções visuais (do Dark Mode vibrante ao Minimalista Amarelo) e implementar efeitos carrosséis angulares, o que manualmente levaria horas.

### Ajustes Manuais
Ajustei manualmente a **curadoria de cores e a densidade de informações**. Intervi para reduzir o ruído visual (removendo o excesso de cores e simplificando ícones) para garantir que a identidade visual estivesse alinhada à sobriedade da Asimov Academy e que a legibilidade fosse prioridade.
